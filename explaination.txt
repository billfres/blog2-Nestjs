

configuration
npm i --save @nestjs/config

install postgres
npm install --save @nestjs/typeorm typeorm postgres(or pg)

after that, we must configure the app.module.ts
 imports: [
    ConfigModule.forRoot({isGlobal: true}),
    TypeOrmModule.forRoot({
      type: "postgres",
      url: '',
      autoLoadEntities: true,
      synchronize: true
    })
  ],
-->On crée le .env et on le remplit
//npm install --save dotenv

-->on crée:
nest g mo src/user/user
nest g co src/user/user
nest g service src/user/user
nest g interface src/user/models

-->on ajoute le imports dans le userModule pour prendre en compte notre model
imports: [
    TypeOrmModule.forFeature([UserEntity])
  ],

-->Ajout du module d'authentification
nest generate module auth

-->Ajout du module jwt
npm install --save @nestjs/jwt

-->on configure par la suite le auth.module

nest generate service auth/auth -->puis on renomme le sous dossier auth en service

-->On installe le module de cryptage
npm i bcrypt --save
(on va l'utiliser dans auth.service: const bcrypt = require('bcrypt'); )
